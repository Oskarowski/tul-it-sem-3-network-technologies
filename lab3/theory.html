• IP Address

    An Internet Protocol address is a unique identificatory of a device or a local network

    Some IP addresses carry special meanings ex: 127.0.0.1 means address of a local computer, 255.255.255.255 means broadcast address

  • Network Subnets

    A subnet is a logical subdivision of an IP network. It allows for the creation of smaller, more manageable networks within a larger network. More efficient (shorter) paths.

  • Network Classes

    Size-based division of networks Each class has a different range of valid IP addresses to accommodate various network sizes.

    Class Number Range of Host Addresses
    A 1.0.0.0 to 127.255.255.255
    B 128.0.0.0 to 191.255.255.255
    C 192.0.0.0 to 223.255.255.255
    D 224.0.0.0 to 239.255.255.255
    E 240.0.0.0 to 255.255.255.255

  • Network Mask

    A netmask is a 32-bit binary mask used to divide an IP address into subnets and specify the network's available hosts.

    In a netmask, two of the possible addresses, the lowest and the highest, are always pre-assigned and unavailable for custom assignment.

  • Routing table

    A routing table is a data table that lists the routes to particular network destinations. It contains information about the available routes and the best paths to reach specific destinations.

  • Default Gateway

    It serves as the exit point for traffic leaving one network and entering another. When a device on a local network wants to communicate with a device on another network, it sends the data packets to the default gateway, which forwards them to the appropriate destination.

  • Broadcast

    Broadcasting in widespread transmission of data to all devices in a network at the same time

  • Subnet Address

    The subnet address is the numerical label assigned to a specific subnet within a larger network. It is obtained by applying the subnet mask to an IP address, effectively isolating the network and host portions

  • Private Network Address

    Private network addresses are reserved IP address ranges designated for use within private networks.
    These addresses are not routable on the public Internet.

  • Network Address Translation (NAT)

    NAT is a technique used to map private IP addresses within a local network to a single public IP address. This allows multiple devices within a private network to share a single public IP address when accessing the Internet
